# Task 1: ML Classification — Telco Customer Churn Prediction

## Overview
Binary classification model to predict customer churn using Logistic Regression and Random Forest.

## Dataset
IBM Telco Customer Churn — loaded automatically in the notebook.

## Setup
pip install pandas numpy scikit-learn matplotlib seaborn joblib

## Run
Open `task1_ml_classification.ipynb` in Google Colab and run all cells.

## Results
| Model | ROC-AUC |
|-------|---------|

| Random Forest | 0.8241 |
